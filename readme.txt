== Description ==

This is simple plugin to display yahoo answers on your website. You can set keywords from admin section to pull the questions and answers from Yahoo.
We have used jquery to view the answers instead of sending user to the site. Every question is listed with corresponding answere.
In the admin section while configuring the widget you can see the preview of the output that helps in deciding the keywords.

== Installation ==

1. Upload `yahoo-ans` to `/wp-content/plugins/` directory
2. Activate plugin through the 'Plugins' menu in WordPress
3. Go to admin section Settings > Yahoo-ans and enter the search query.
4. Set the widget from admin section.
